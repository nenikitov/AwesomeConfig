# Pop-ups

## Brightness

- Shows
    - When brightness is changed
    - Left top corner of the screen
- Contains
    - Icon
    - Brightness %
    - Vertical slider
- Mock
    ```
    ╭────╮
    │ 69 │
    │ ╭╮-│
    │ ││-│
    │ ││-│
    │ ▐▌-│
    │ ▐▌-│
    │ ▝▘-│
    │ 󰃟  │
    ╰────╯
    ```
- Interactivity
    - Slider is clickable and scrollable to change brightness with the mouse
    - Icon changes depending on the current brightness
